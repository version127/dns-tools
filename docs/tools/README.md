# Tool documentation

Each tool owns one folder:

- [DNS Lookup](dns-lookup/)
- [DNS Trace Explorer](dns-trace/)
- [DNS Change Checker](dns-change-checker/)
- [Nameserver Delegation Checker](nameserver-checker/)
- [DNSSEC Chain Checker](dnssec-checker/)
- [SOA Consistency Checker](soa-checker/)
- [CAA Policy Checker](caa-checker/)

Keep tool-specific behavior, limitations, result meaning, and maintenance notes in its folder. Shared transport, privacy, security, API, and self-hosting information belongs in the top-level `docs` files.
