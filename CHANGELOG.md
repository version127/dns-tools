# Changelog

## 1.0.0 — 2026-07-15

First public-ready release of DNS Tools.

- Added DNS Lookup, DNS Trace Explorer, DNS Change Checker, Nameserver Delegation Checker, DNSSEC Chain Checker, SOA Consistency Checker, and CAA Policy Checker.
- Added normalized DNS results, raw per-query evidence, CSV downloads where useful, favicon display when available, resolver and authoritative lookup modes, and ASN enrichment.
- Added bounded DNS transports, public-target checks, weighted rate limiting, no-store API responses, and same-origin browser protection.
- Added a self-hosted Next.js application with Docker support, health checks, API documentation, an OpenAPI contract, and indexing disabled by default.
